# Calculator

This is my implementation of The Odin Project's Calculator project from the Foundations Course.

## Project Description

A browser-based calculator that performs basic arithmetic operations. Features operation chaining, decimal support, and a clean interface styled after the iOS calculator. The calculator evaluates expressions as you type and displays results in real-time.

## Skills Demonstrated

- DOM manipulation with JavaScript
- Event listeners (button clicks)
- Functions and switch statements
- Conditional logic for calculator operations
- String manipulation and parsing
- State management with multiple variables
- Input validation and error handling
- CSS Flexbox for layout
- Responsive button styling

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Operation chaining (12 + 7 - 1 evaluates correctly)
- Decimal number support with automatic rounding
- Clear button to reset calculator
- Backspace button to delete last input
- Division by zero error handling
- Real-time display updates
- Prevents invalid operations (consecutive operators, incomplete expressions)
- Starts fresh calculation after displaying a result

## How to Use

1. Click number buttons to enter values
2. Click an operator (+, -, \*, /) to select operation
3. Enter the second number
4. Click = to see the result, or click another operator to chain operations
5. Use C to clear everything
6. Use <= to backspace and delete the last character
